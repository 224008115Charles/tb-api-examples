Teambition API - Python 调用示例
================================

## 示例内容

该示例中，包含以下接口调用
- 创建项目
- 创建任务
- 完成任务
- 创建日程
- 更新日程标题
- 添加成员
- 移除成员
- 删除任务
- 删除日程
- 删除项目

## 运行

```bash
pip install -r requirements.txt
python flask_example.py
```

然后访问http://localhost:3000/auth
