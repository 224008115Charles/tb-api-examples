Teambition API - Postman事例
================================

## 事例内容

该事例中，包含以下接口调用
- 创建项目
- 获取项目任务列表
- 创建任务
- 完成任务
- 创建日程
- 更新日程标题
- 添加成员
- 一处成员
- 删除任务
- 删除日程
- 删除项目
- 获取accesstoken

## 运行
- 在Postman中Import该文件
- 在Colletion中进入对应的Teambition Demo
